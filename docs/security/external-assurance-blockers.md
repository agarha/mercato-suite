# External Security and Compliance Blockers

Status: blocked by vendor, legal, or production access.

## Pentest

Required evidence:

- signed statement of work
- tested environment URL
- final report
- remediation report for high/critical findings

Current status: blocked until a pentest vendor and target staging environment are provided.

## SOC 2 Type II

Required evidence:

- auditor engagement
- control matrix
- evidence repository
- observation period dates

Current status: blocked until an auditor is selected and company controls are available.

## SIEM Integration

Required evidence:

- selected SIEM
- log forwarder configuration
- alert routing
- test alert acknowledgement

Current status: local audit and metrics exist; production SIEM target is not available.

## Zero-Retention AI Contracts

Required evidence:

- provider contract
- data retention terms
- DPA/security review

Current status: AI Copilot is deferred from MVP and contracts are not available.
